1. Program wymaga pythona wersji 3.9+
2. Uruchamianie (z folderu src): python main.py
3. Dostępne argumenty:
	-l, --length [int]	Wybranie długości sygnału
	-a, --amount [int]	Wybranie ilości sygnałów do wysłania
	-n, --noise [float]	Wybranie mocy szumu (zakres 0-1)
	-s, --show		Wyświetlenie ilości powtórzeń dla każdego sygnału	
	-r, --repeat [int]	Wybranie ilosci powtorzen symulacji
